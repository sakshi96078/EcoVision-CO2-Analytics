# EcoVision-CO2-Analytics
# 📊 CO₂ Emissions Data Analysis



---

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on global CO₂ emissions data to identify trends, major contributors, and relationships with economic and population factors.

The objective is to transform raw environmental data into **meaningful insights** for sustainable decision-making.
